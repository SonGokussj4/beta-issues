What does this app do?

This is an issue tracker specific for ANSA/META software from BETA CAE company.

BETA rolls software updates once in a month/two with PDFs containing resolved issues:

  • ANSA_release_notes.pdf, META_release_notes.pdf, ...

These contain fields of resolved incidents in format:

  • [ Resolved Incident: ANSA-16523], [ Resolved Incident: #4635, META-31523], and so on...

The purpose of this WebApp is to:

  • (these PDF files),
  • which saves resolved issues into database
  • and compares them with our pending issues (added manually in the page)
  • inform user about new resolved issues (if any)
  • automatically update tables of Un/Resolved issues in page
